# Images Directory

Place your hero banner image here.

## Hero Banner Image

To add a hero banner image to the homepage:

1. Place your image file in this directory (e.g., `hero-banner.jpg`)
2. Update `content/_index.md` and uncomment the hero image line:
   ```toml
   [hero]
     image = '/images/hero-banner.jpg'
     alt = 'GPEML Conference'
     showText = true  # Optional: show text overlay on image
     title = 'GPEML Conference'
     subtitle = 'Pacific Rim International Conference on Artificial Intelligence'
   ```

## Recommended Image Specifications

- **Width**: 1400px or wider
- **Height**: 400-600px (recommended aspect ratio: 16:9 or 21:9)
- **Format**: JPG, PNG, or WebP
- **File size**: Optimize for web (under 500KB recommended)

## Image Sources

You can use:
- Conference venue photos
- City/venue skyline
- Abstract AI/ML themed images
- Conference branding graphics
